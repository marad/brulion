## Approach (whole spec)

Pure UI: `index.html` markup + `styles.css` for the hero, a small tested view-swap
helper in `ui.ts`, and rewiring in `main.ts`. No file-behavior code is touched.

### Markup (`index.html`)
- Header: add a persistent `<span class="wordmark">Brulion</span>` first. Keep
  `#install-app` (FEAT-0030) and the `#toggle-sidebar` / `#toggle-vim` toggles
  (still `hidden`). Add `<button id="reopen-folder" type="button" hidden>Open
  folder</button>` (post-folder re-pick).
- Move `#open-folder` and `#resume-access` OUT of the header INTO a new
  `<section id="welcome">` (the hero): a `Brulion` heading, a pitch line, a
  file-fidelity line, then the **Open folder** button + the (`hidden`) **Resume
  folder access** button. The section sits inside `main.workspace` alongside
  `#editor`, shown by default.

### View swap (`ui.ts`, unit-tested)
`export function showWorkspace(refs: { welcome; sidebar; toggleSidebar; toggleVim;
reopen }): void` — sets `welcome.hidden = true` and the other four `.hidden =
false`. One place for the pre→post folder flip (AC-2). Initial HTML state is the
inverse (welcome shown; sidebar/toggles/reopen hidden) → AC-1/AC-3.

### Rewiring (`main.ts`)
- Query `#welcome`, `#reopen-folder`, add to the mount null-check.
- In `onListChanged`, replace the three manual `hidden=false` lines with
  `showWorkspace({...})` (welcome + sidebar + both toggles + reopen). Install stays
  governed by FEAT-0030 only.
- Wire re-pick: `wireOpenFolder(reopenButton, resumeButton, openNote)` (reuses the
  exact open flow as the welcome CTA) — AC-5. The welcome CTA keeps its existing
  `wireOpenFolder(openButton, resumeButton, openNote)` + `restoreFolder(...)`.

### Styles (`styles.css`)
`#welcome` centered column (flex, max-width, generous spacing), reusing the
existing palette (`#1a1a1a` text, accent `#9a3b2f` for the wordmark/CTA emphasis,
`#6a6a6a` for the muted file-fidelity line). `.wordmark` a quiet brand label in the
header. `header button` styling is reused for the CTA; the primary CTA may get a
slightly emphasized look via an extra class.

### Tests
