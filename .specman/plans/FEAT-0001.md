## AC-4 (added) — The app is live at the GitHub Pages URL

Approach: Not a deterministic local command — the URL only exists after the
first successful Actions run, and an external fetch is flaky as a seal gate.
Verified **out-of-band post-deploy**: after merge + a green Actions run, fetch
`https://marad.github.io/brulion/` and assert HTTP 200, the document title

source is done by the agent via `gh`, not manually:
`gh api --method POST repos/marad/brulion/pages -f build_type=workflow`
(falls back to `--method PUT` if the Pages site already exists).
