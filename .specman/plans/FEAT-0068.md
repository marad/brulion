## Implementation

Presentation-only; the bulk is `src/styles.css`, with three small JS hooks. Three
phases, all under this one spec (see DECISIONS.md / M34.md for the right-sizing).

### P1 — Motion foundation + overlay enter/leave (AC-1..5)
- `src/styles.css` `:root`: add `--motion-fast: 0.001s`, `--motion-medium: 0.001s`,
  `--motion-slow: 0.001s`, `--ease`, `--ease-out`. Add `:root.motion-ready` resolving the
  three durations to real values (~120/190/260ms). Add
  `@media (prefers-reduced-motion: reduce) { :root.motion-ready { … : 0.001s } }`.
- `src/main.ts`: after first paint, `requestAnimationFrame(() =>
  requestAnimationFrame(() => document.documentElement.classList.add("motion-ready")))`.
  Extract a tiny `markMotionReady(root?)` helper so it is unit-testable.
- `src/styles.css`: backdrops (`#switcher-backdrop, #palette-backdrop,
  #conflict-backdrop, #settings-backdrop`) — opacity transition + `display allow-discrete`
  + `@starting-style { opacity: 0 }`. Inner dialogs (`#switcher, #palette, #conflict,
  .settings-dialog`) — opacity + `translateY`/`scale` transition + `@starting-style`.
  `#welcome` — fade in. Context menu / autocomplete popup — `@starting-style` enter.

### P2 — Structural motion (AC-6..8)
- `src/ui.ts` `wireSidebarResize`: add `sidebar.classList.add("resizing")` on pointerdown,
  remove on drag end. New `ui.test.ts` case asserting the class toggles across the drag.
- `src/styles.css`: `#sidebar` `transition: flex-basis, opacity`; `#sidebar.resizing
  { transition: none }`; change `.workspace.sidebar-collapsed #sidebar` from
  `display:none` to `flex-basis:0; opacity:0; pointer-events:none` (+ `overflow:hidden` on
  `#sidebar`). Mobile drawer: `translateX` transition + `@starting-style`; backdrop
  cross-fade. `.folder-children` enter fade/rise via `@starting-style`.

### P3 — Micro-interactions + theme cross-fade (AC-9..11)
- `src/styles.css`: `transition: background-color/color/border-color var(--motion-fast)`
  on buttons, `.note-row`, `.note-name`, `.note-delete`, `.folder-header`, switch/palette
  rows, active highlights. Theme cross-fade: transition `background-color/color/border-color`
  on `body, header, #sidebar, .workspace` and key panels (gated by `motion-ready` via the
  token, so the async initial apply doesn't flash).
- `src/selection-toolbar.ts`: replace `el.style.display = "flex"/"none"` with
  `el.hidden = false/true`; add `.cm-selection-toolbar[hidden] { display:none }` and its
  enter animation in CSS.

### Moat (AC-12)
No `.md` writes; no transition on any `.cm-*` content node (only chrome). Verified by
review + the unchanged editor/rendering e2e.
